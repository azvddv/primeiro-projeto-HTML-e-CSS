Este projeto consiste na criação de uma página de login para um site de café
desenvolvida utilizando apenas HTML e CSS.
O objetivo é oferecer uma interface moderna, elegante e intuitiva, 
alinhada à identidade visual de cafeterias,
utilizando tons terrosos, tipografia acolhedora e um layout limpo.
A página inclui um formulário de autenticação, contendo campos de e-mail e senha, além de botões estilizados para entrar e recuperar senha.
